# 🎮 Hangman Game in Python

This project is a simple **text-based Hangman Game** developed using Python as part of my **CodeAlpha Internship**.

---

## 📌 Project Description

The Hangman game is a word guessing game where:
- A random word is selected
- The user guesses one letter at a time
- The user has limited attempts (6 chances)

---

## 🚀 Features

- Random word selection
- User input handling
- Limited attempts
- Win and Lose conditions
- Simple terminal-based game

---

## 🛠️ Technologies Used

- Python
- Concepts used:
  - Loops
  - If-else conditions
  - Lists
  - Strings
  - Input/Output

---
